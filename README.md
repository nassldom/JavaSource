## Wie du dieses Repository nutzen kannst

Dieses Repository folgt der Struktur von *Introduction to Computer Science using Java* und ist in Parts (1–14) entsprechend der Online-Kapitel organisiert.  
Du kannst die Beispiele auf zwei Arten nutzen:

1. **Parallel zum Online-Kurs**  
   - Öffne das jeweilige Part/Kapitel im Kurs (z. B. Part 7 – Arrays, Kapitel 60–68).  
   - Suche in der README nach dem entsprechenden Part und Kapitel.  
   - Öffne die dort aufgeführten `.java`‑Dateien in deiner IDE und experimentiere mit dem Code (ändern, erweitern, Debuggen).

2. **Als eigenständigen Lernpfad**  
   - Starte bei **Part 2** und arbeite dich der Reihenfolge nach bis Part 14 durch.  
   - Lies jeweils zuerst die Kommentare bzw. den Code, dann führe das Programm aus und variiere Eingaben und Implementierung.  
   - Nutze den Abschnitt **„Mapping zum Online-Kurs“**, um bei Bedarf die passende Theorie‑Seite im Originalkurs zu finden.

Empfohlene Umgebung: **Java 17+** und eine IDE wie IntelliJ IDEA, Eclipse oder VS Code mit Java‑Extension.  
Die Beispiele sind bewusst klein gehalten, damit du Schritt für Schritt Sprache, OOP‑Konzepte und JavaFX lernen kannst.
